
# VAmPI — Beginner-Friendly API Security Walkthrough## Prerequisites
Before getting started, install the following tools:

1. Postman

If Postman is not installed, refer to this guide for installation on Linux:  
https://faranheit.medium.com/postman-installation-on-linux-e293c399ed4d

2. Burp Suite

Install Burp Suite Professional using the following repository:
https://github.com/xiv3r/Burpsuite-Professional
Complete the setup by referring to the following video guide:
https://youtu.be/gVaXlwZ6qrI?si=T-3vAMHzUE0vry7W
